Sobre Mim

Olá! Seja bem-vindo à minha página de fãs de Rick and Morty. Criei este projeto para explorar e compartilhar informações sobre os personagens, episódios e lugares desse universo tão divertido e caótico que eu adoro.

Utilizo a Rick and Morty API para trazer dados atualizados e precisos sobre os personagens, episódios e muito mais.

Tecnologias que Usei

  • Angular 20 com Standalone Components
  • REST API para integração de dados
  • Vite para build e servidor de desenvolvimento

Sobre este Projeto

Este é um projeto pessoal, criado com fins educativos e como uma forma de praticar minhas habilidades de desenvolvimento. Não tenho nenhuma relação comercial com a marca Rick and Morty — todos os direitos pertencem aos seus criadores.

Espero que você aproveite a navegação e descubra algumas curiosidades sobre seus personagens favoritos!
